HTML5 - Hyper Text Markup Language (<head> content </head>)

Self closing tags - hr, br, meta, img

===> It is Used to create a webpage
===> It is NOT a Programming Language
===> It is Markup Language
===> Every website can start with HTML5

1) Heading - h1 to h6

2) Paragraph - p

3) Hr

4) Break tag - br

5) Text Formatting - b or strong, i, u, sup, sub and mark

6) Anchor Tag (Linking) - a

7) Image Tag - img

8) List - Ordered list and Unordered list

9) Tables - tr, th, td
